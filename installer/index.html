WebSocket URL


De WebSocket URL is de verbinding via welke de biertap bericht krijgt van een geslaagde betaling. De betaling wordt verwerkt door LNbits via een LNURLdevice. Om een WebSocket URL te krijgen, doorloop de volgende stappen:

  1. Maak een wallet aan op de demo server van LNbits (https://legend.lnbits.com/).
  2. Schakel de LNURLDevice extension on (Extensions onder het kopje Extensions in LNbits).
  3. Kies onder LNURLDevice in LNbits, en maak een nieuw LNURLDevice aan.
  4. Onder de LNURLDevice settings staat de WebSocket URL. Vul deze hierboven in.
